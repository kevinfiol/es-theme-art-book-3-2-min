# es-theme-art-book-3-2-min

A modified version of [this old](https://github.com/anthonycaccese/es-theme-art-book-3-2) EmulationStation theme.

Made to resemble a cross between MinUI and AnalogueOS. Uses [analogue-os-font](https://github.com/AbFarid/analogue-os-font).

## ArkOS Loading Screen
Put `loading.jpg` in your `EASYROMS > launchimages` folder.
